# FluxMoment - 情感驱动的 AI 灵感捕捉系统

FluxMoment 是一款融合了极致视觉美学与强悍 AI 自动化能力的个人记忆库。它不仅记录事实，更通过 AI 深度解析、情感识别与多维展示，将你的碎片信息沉淀为有温度的数字记忆。

## 🏗️ 技术架构 (Technical Architecture)

### 1. 全端全场景接入 (Ubiquitous Access Layer)
- **随时随地，极简捕捉**：打破应用孤岛，支持通过 **Web 网页端** 与 **外部 AI 助手 (如 龙虾/OpenClaw)** 两种入口并行录入，确保灵感在迸发的瞬间即被捕获。
- **OpenClaw Skill 驱动**：内置标准化 **MCP Server** 接口，通过调用专门的“记忆 Skill”，你可以直接在 AI 对话框中输入片段，无需切换窗口，真正实现“对话即记录”。
- **同构逻辑核心 (`aiLogicCore.ts`)**：采用核心算法底座，确保 Web 端与 MCP 接入层在处理用户意图、情感识别、标题提炼算法上完全一致，维持数据的高质量与统一性。

### 2. AI 智能引擎 (Intelligence Engine)
- **多级分发模型**:
  - **Grade A (核心事实)**: 采用 `[核心实体 + 状态]` 极简提炼算法（如：“部署完成” -> “发布应用”）。
  - **Grade B (感性时刻)**: 侧重具象短句描述与情感共鸣。
  - **Grade C (碎片闪念)**: 高效过滤噪声，归拢瞬时火花。
- **智能话题合并策略**: 实时检索历史 Thread，基于语义重合度与时间窗口（1小时内）自动执行追加或新建决策。
- **情感识别系统**: 自动识别 `happy`, `focused`, `tired`, `adventurous` 等模式，并映射至 50 套手工精选的 3D 图标库 (`avatarVariant`)。
- **向量语义搜索**: 集成向量嵌入 (Embedding) 接口，支持混合动力搜索：标签硬过滤 + 关键字匹配 + 语义余弦相似度 + AI 二次重排 (Rerank)。

### 3. 数据与同步层 (Data & Sync)
- **Google 账号一键登录**：基于 Firebase Auth 的安全验证，**用户需要登录 Google 账号**以确保记忆在多端实时同步，并获得私密及持久的数据存储。
- **实时后端**: Google Firebase Firestore (NoSQL)，实现多端毫秒级同步。
- **本地增强**: 
  - **乐观更新策略**: 写入操作秒级响应，后台异步同步，确保极致流畅感。
  - **双层缓存模型**: 基于 `localStorage` 的 `cacheFingerprint` 机制，实现 AI 总结数据的离线加载与瞬时刷新。
  - **混合媒体管理**: 基于 Firebase Storage 的流式媒体处理，支持图片与视频的瀑布流预览。

## 💻 核心逻辑细节

### AI 处理流 (Classification Flow)
1. **Input**: 用户自然语言输入（Text/Media）。
2. **Context Retrieval**: 自动拉取最近 1 小时的历史话题上下文。
3. **LLM Node**: 构建包含 Few-shot 示例的结构化 Prompt，强制 LLM 返回符合规范的 JSON。
4. **Validation**: 对返回的话题 ID、图标编号进行二次校验与回退（Fallback）。
5. **Persistence**: 计算内容 Hash 生成唯一 ID，并行执行数据库写入与向量生成。

### 视觉特性 (Visual Excellence)
- **Micro-animations**: 基于 `framer-motion` 的物理级动画（弹簧系数：stiffness 400, damping 35）。
- **Frosted Glass (Glassmorphism)**: 采用高斯模糊（blur 8px）配合饱和度增强（saturate 130%）的卡片设计。
- **Responsive Media Mosaic**: 智能媒体拼贴组件，根据附件数量自动计算布局比例。

## 🚀 开发者指南
