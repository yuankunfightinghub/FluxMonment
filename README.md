# FluxMoment - 情感驱动的 AI 灵感捕捉系统

FluxMoment 是一款融合了极致视觉美学与强悍 AI 自动化能力的个人记忆库。它不仅记录事实，更通过 AI 深度解析、情感识别与多维展示，将你的碎片信息沉淀为有温度的数字记忆。

---

## ✨ 产品理念：美学与理性的交织

- **极简记录，深度沉淀**：通过 Grade A/B/C 三级分发策略，实现从“随手一记”到“系统整理”的无缝转化。
- **情感映射**：不仅仅是文字，系统通过识别 10 种微妙情绪，为每一条记录赋予独特的视觉色彩与图标。
- **Fluid Chronos**：电影感的时间轴布局，让记忆像流沙一样自然流淌。
- **场景共鸣**：内置由顶级产品运营精选的示例卡片，涵盖深夜办公、独处宁静与家庭温情，激发创作灵感。

## 🏗️ 技术架构 (Technical Architecture)

### 1. 全端全场景接入 (Ubiquitous Access Layer)
- **随时随地，极简捕捉**：支持通过 **Web 网页端** 与 **外部 AI 助手 (如 龙虾/OpenClaw)** 并行录入。
- **对话即记录**：内置标准化 **MCP Server** 接口，直接在 AI 对话框中输入片段，无需切换窗口。
- **快捷交互**：输入框支持快捷键操作，单按 `Enter` 换行，`Cmd + Enter` (Mac) 或 `Ctrl + Enter` (Win) 极速提交。

### 2. AI 智能引擎 (Intelligence Engine)
- **多级分发模型**: 采用 `[核心实体 + 状态]` 极简提炼算法，精准归拢瞬时火花。
- **智能话题合并策略**: 实时检索历史 Thread，基于语义重合度与时间窗口自动执行决策。
- **向量语义搜索**: 集成 Embedding 接口，支持混合动力搜索：标签过滤 + 关键字匹配 + 语义余弦相似度 + AI Rerank。

### 3. 数据与极致同步 (Data & Seamless Sync)
- **零门槛预览 (Guest Mode)**: 
  - 未登录状态下，用户可直接查看精选示例卡片，感受应用的核心价值。
  - 支持 **LocalStorage 冗余存储**，确保未登录时的输入也能被即时保存。
- **智能登录拦截**: 只要用户尝试提交第一条记录，系统会立即触发 Google 安全登录逻辑，确保数据私密。
- **无缝数据迁移**: 登录成功后，系统会自动检测并**将本地缓存的 Moment 批量迁移至云端 Firestore**，确保记忆永不丢失。
- **实时后端**: Google Firebase 服务，实现多端毫秒级同步及乐观更新策略。

## 💻 核心逻辑细节

### 交互中心 (UX Center)
- **智慧交互**: `InputCapsule` 实时推断话题主题，Hover 发送按钮即可见快捷键提示。
- **权限感知渲染**: 根据登录状态动态控制底部的 `Moments` 与 `Daily Memory` 标签页，保持非登录态的界面的纯净。

### 视觉特性 (Visual Excellence)
- **Micro-animations**: 基于 `framer-motion` 的物理级动画（弹簧系数：stiffness 400, damping 35）。
- **Frosted Glass (Glassmorphism)**: 采用高斯模糊配合饱和度增强的卡片设计。
- **Responsive Media Mosaic**: 智能媒体拼贴组件，根据附件数量自动计算布局比例。
