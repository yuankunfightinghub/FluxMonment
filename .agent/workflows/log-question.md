# /log-question 命令

当用户输入此命令时，无论提问中是否包含 savePrompt，请立即执行以下操作：

1. **提取内容**：如果用户在命令后跟随了文字（例如 `/log-question 今天天气不错`），则提取该文字；如果只有命令本身，则提取上一轮次的提问原文。
2. **存档记录**：
   - 打开 `/Users/yuankun/antigravity/question.md`。
   - 在顶部插入当前日期、会话名称、会话 ID 和提问原文。
3. **反馈确认**：完成后告知用户“提问已成功录入档案库”。
